## Dsend

Dsend is a distributed message queue system.

### Current Architecture

```text
                           ┌─────────────────────┐
                           │      Producers      │
                           └──────────┬──────────┘
                                      │
                                      ▼
                    ┌─────────────────────────────────┐
                    │            Broker               │
                    │                                 │
                    │  ┌───────────────────────────┐  │
                    │  │        Ready Queue        │  │
                    │  │   (Circular Buffer)       │  │
                    │  └─────────────┬─────────────┘  │
                    │                │                │
                    │                ▼                │
                    │  ┌───────────────────────────┐  │
                    │  │     Delivery Manager      │  │
                    │  │  Generates Ack Tokens     │  │
                    │  └─────────────┬─────────────┘  │
                    │                │                │
                    │                ▼                │
                    │  ┌───────────────────────────┐  │
                    │  │      InFlight Store       │  │
                    │  │ token -> delivery state   │  │
                    │  └─────────────┬─────────────┘  │
                    │                │                │
                    │      timeout   │   ack(token)   │
                    │                │                │
                    │                ▼                │
                    │  ┌───────────────────────────┐  │
                    │  │    Redelivery Manager     │  │
                    │  └─────────────┬─────────────┘  │
                    │                │                │
                    │     retry      │     maxRetry   │
                    │                ▼                │
                    │  ┌───────────────────────────┐  │
                    │  │    Dead Letter Queue      │  │
                    │  └───────────────────────────┘  │
                    └─────────────────────────────────┘
                                      │
                                      ▼
                           ┌─────────────────────┐
                           │      Consumers      │
                           └─────────────────────┘
```
